# 基于javaweb的珠海科技学院教务管理系统（JAVA Web期末大作业）

这是一款 Spring Boot + MyBatis + Shiro + JSP 的Java全栈项目，偏向传统Java Web开发模式

## 技术栈

- Spring Boot
- MyBatis
- Shiro
- JSP

## 项目结构

```
EducationalManagementSystem/
├── .idea/   
├── .mvn/
├── sql/
├── src/

│   │   ├── java/
│   │   ├── resources/
│   │   └── webapp/
│   └── test/
│       └── java/
├── target/
├── EducationalManagementSystem.iml
├── pom.xml
└── README.md
```
